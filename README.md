# Markowitz Portfolio Optimization Model

This project builds an Excel-based portfolio optimization model using Modern Portfolio Theory and Markowitz portfolio optimization. The model analyzes a diversified set of assets using historical monthly returns, risk-return statistics, covariance, correlation, and Sharpe ratio calculations.

The goal of this project is to compare different portfolio construction approaches and identify portfolios with improved risk-adjusted performance.

## Project Overview

Portfolio optimization is used to evaluate how investors can allocate capital across assets to balance expected return and risk. In this project, I constructed and analyzed portfolios using historical asset return data and Markowitz optimization concepts.

The project compares:

* Equal-weight portfolio
* Minimum-variance portfolio
* Maximum-Sharpe portfolio
* Investment opportunity set

The analysis supports portfolio decision-making by showing how different asset allocations affect expected return, standard deviation, and risk-adjusted performance.

## Methodology

The model follows a structured portfolio optimization process:

1. Collected historical monthly return data for a diversified set of assets.
2. Calculated asset-level summary statistics, including mean returns and standard deviations.
3. Built variance, covariance, and correlation matrices to measure relationships between assets.
4. Constructed an equal-weight portfolio as a baseline.
5. Calculated the minimum-variance portfolio to identify the lowest-risk allocation.
6. Calculated the maximum-Sharpe portfolio to identify the portfolio with the strongest risk-adjusted return.
7. Visualized the investment opportunity set to compare portfolio risk and return combinations.

## Key Concepts Used

| Concept                    | Description                                                           |
| -------------------------- | --------------------------------------------------------------------- |
| Expected Return            | Estimated average return based on historical monthly return data.     |
| Standard Deviation         | Measures the volatility or risk of each asset and portfolio.          |
| Variance                   | Measures dispersion of returns around the mean.                       |
| Covariance                 | Measures how two assets move together.                                |
| Correlation                | Standardized measure of the relationship between asset returns.       |
| Sharpe Ratio               | Measures excess return per unit of risk.                              |
| Minimum-Variance Portfolio | Portfolio allocation with the lowest overall volatility.              |
| Maximum-Sharpe Portfolio   | Portfolio allocation with the highest risk-adjusted return.           |
| Investment Opportunity Set | Visual representation of possible portfolio risk-return combinations. |
